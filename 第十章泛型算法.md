### 10.3.3 lambda的捕获和返回
```
当向一个函数传递lambda时售后机定义了一个新类型和新对象
传递的参数就是编译器为命名的对象

默认情况下lambda 生成类毒包含一个对应的lambda所捕获的数据成员
lambda的数据成员在lambda对象创建时被初始化
```
#### 值捕获
```
[v1]{}
值捕获采用的拷贝变量,值拷贝的实际是在lambda对象创建的时候拷贝
```
#### 引用捕获
```

note:
当以引用方式捕获变量时必须保证执行引用时对象必须存活

建议:尽量保持lambda的简单化
```
